Scene - camera - renderer & lights!

We behandelen eerst enkele basisbegrippen die steeds zullen terugkeren.
Zorg er voor dat je deze basiselementen zeker begrijpt, experimenteer ook eens met enkele alternatieve materialen en lichten.

Het is een goed idee om overzicht te bewaren van de variabelen die je zal gebruiken.
Zorg voor een dom-element waarin je de content wil plaatsen. In ons geval is dit steeds een div met id 'container'. 

    
        // waarin zullen we de content plaatsen?
        // pro-tip: zorg er voor dat je container een hoogte heeft, anders zal er niet veel te zien zijn.

        var $container = $('#container');
        var WIDTH = $container.width(),
            HEIGHT = $container.height();

        // variabelen

        var scene, camera, renderer, object, light;

Scene - camera - renderer

Het spreekt voor zich, de scene is de omgeving waarom we alle volgende elementen zullen plaatsen. 

    
        // opzetten van de scene

        scene = new THREE.Scene();

Zo simpel is het! :)
Alleen zal er nog niet veel te zien zijn. We moeten onze omgeving nog renderen en we hebben nog iets nodig om er naar te kijken... 

    
        // eerst onze renderer, we kiezen voor de WebGLRenderer.
        // waarom niet de SVG- of CanvasRenderer?

        renderer = new THREE.WebGLRenderer();

        // de camera heeft enkele parameters nodig

        var FOV = 45,               // Field Of View, hoe groter de kijkhoek: hoe meer beeld
            ASPECT = WIDTH/HEIGHT,  // dit is de ratio van onze container
            NEAR = 0.1,             // het dichtste vlak van het kijk-frustrum
            FAR = 10000;            // het verste vlak van het kijk-frustrum

        // we kiezen voor de PerspectiveCamera, welke bestaan er nog?

        camera = new THREE.PerspectiveCamera(FOV,ASPECT,NEAR,FAR);

Zoals gezegd: alles moet op de scene! Ook de camera.

    
        // camera toevoegen aan de scene

        scene.add(camera);

        // camera positioneren, nieuwe objecten staan standaard op coördinaat 0,0,0

        camera.position.z = 300;

Laatste stap! Renderen!

    
        // grootte instellen, zelfde als onze container.

        renderer.setSize(WIDTH,HEIGHT);

        // resultaat toevoegen aan de domcontent van onze container.

        $container.append(renderer.domElement);

        // renderen!

        renderer.render(scene, camera);

tadaaa!

... er valt natuurlijk nog niets te zien!

Object aanmaken

We maken een simpele bol (sphere).
Experimenteer zelf eens met enkele verschillende materialen en parameters!

  • MeshBasicMaterial
  • MeshLambertMaterial
  • MeshNormalMaterial
  • MeshPhongMaterial
  • ...
    
        // object aanmaken

        var radius = 60, segments = 16, rings = 16;
        var sphereGeometry = new THREE.SphereGeometry(radius,segments,rings);

        var materialParameters = {
            color:0xffee55,
            reflectivity: 1
        };

        var sphereMaterial = new THREE.MeshLambertMaterial(materialParameters);

        object = new THREE.Mesh(sphereGeometry,sphereMaterial);

        scene.add(object);

En lights!

AmbientLight, DirectionalLight, HemisphereLight, Spotlight... wij kiezen hier voor PointLight.

    
        // lights!

        light = new THREE.PointLight(0xffffff);
        light.position.x = 10;
        light.position.y = 50;
        light.position.z = 130;

        scene.add(light);

Nooit vergeten... Renderen!

    
        renderer.render(scene, camera);

Voila!
