Voor we beginnen

Doel van deze tutorial

We gebruiken three.js om cross-platform browsertoepassingen te maken voor google cardboard.

Op het einde van deze tutorial

  • heb je ervaring met het aanmaken en animeren van 3D objecten via three.js
  • ben je in staat om een equirectangular panorama te genereren
  • kan je gebruik maken van enkele three.js controls, in het bijzonder de accelerometers van mobiele toestellen
  • kan je een simpele navigatie interface opzetten voor google cardboard toepassingen
  • kan je 3D objecten importeren via three.ColladaLoader()

Benodigdheden

  • Google cardboard
  • three.js inclusief examples
  • basiskennis javascript (optioneel ook jquery) en html
  • noties van 3D begrippen (shaders, vertex, rederer, meshes, bitmap...)

Wat is three.js

Three.js is een open source javascript library die ons in staat stelt om via WebGL complexe 3D afbeeldingen en animaties te maken en te visualiseren zonder nood aan extra plugins. Dit maakt three.js in grote mate browser- en platform onafhankelijk.

WebGL maakt bovendien gebruik van de GPU wat een enorme impact heeft op de snelheid. Ook canvas en svg renderers zijn beschikbaar, maar worden in deze tutorial niet besproken.

Voor een overzicht van alle beschikbare objecten, shaders, renderers... raden we sterk aan om de documentatie op threejs.org of github te bekijken.
